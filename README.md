# Fruit Pop 2D

A colorful HTML5 arcade fruit game built with Canvas and JavaScript.

## Features

- Smooth particle effects
- Combo scoring system
- Bomb obstacles
- Responsive layout
- Web Audio sound effects
- GitHub Pages ready

## Play Online

https://kurotsmile.github.io/fruit-pop-2d/

## Controls

- Click or tap fruits to pop them
- Avoid bombs
- Chain combos for higher scores

## Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- CSS3
